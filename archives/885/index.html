Homebrew HEAD formula upgrade

brew 中可以使用 --HEAD 来从最新的源代码安装, 但是在升级时这个就有了问题. 因为使用 Git 或者别的 VCS 拉下来的源代码是没有版本号的, 这样就无法得知软件是否更新.

但是 brew 是支持单个 HEAD 软件升级的, 使用 brew upgrade --fetch-HEAD formula 就可以. 那么简单来说只需要 xargs 一下就行.

 brew ls|xargs brew upgrade --fetch-HEAD
