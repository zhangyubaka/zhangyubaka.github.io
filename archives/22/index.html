如何在Openshift上安装FFMPEG

Oct 29, 2014

因为里区要推出在线看片功能…所以研究了一下

首先,你需要安装rhc(这个大家应该都安装了吧

然后打开Git bash,键入以下指令(先rhc ssh

cd $OPENSHIFT_DATA_DIR
mkdir bin
wget http://www.tortall.net/projects/yasm/releases/yasm-1.2.0.tar.gz
wget http://ffmpeg.org/releases/ffmpeg-2.0.1.tar.gz

tar -xvf yasm-1.2.0.tar.gz
cd yasm-1.2.0
./configure --prefix=$OPENSHIFT_DATA_DIR/bin --bindir=$OPENSHIFT_DATA_DIR/bin
make
make install
export PATH=$OPENSHIFT_DATA_DIR/bin:$PATH

cd $OPENSHIFT_DATA_DIR
tar -xvf ffmpeg-2.0.1.tar.gz
cd ffmpeg-2.0.1
./configure --prefix=$OPENSHIFT_DATA_DIR/bin --bindir=$OPENSHIFT_DATA_DIR/bin
make
make install

换新版本的话只要替换url即可,另外一定到准备好空间,不然分分钟玩脱
