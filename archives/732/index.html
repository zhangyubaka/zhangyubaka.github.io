Homebrew 无法更新

Aug 20, 2016

Homebrew 最近几天都没更新了, brew update 一直都是 Already up-to-date.

哈?肯定有问题.敲了眼 https://github.com/homebrew/brew 然后发现的确有问题.

执行

cd $(brew --repo); git fetch; git reset --hard origin/master; brew update

即可.

估计是 revert 什么吧…
