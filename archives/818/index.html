Livepatch

这是 Ubuntu 的 snap 黑科技, 可以不重启更新内核.

启用黑简单, 首先去 https://ubuntu.com/livepatch 申请一个 Token .

Token 是一个 32 位的字符串, 到手之后就可以配置了.

sudo apt install snapd
sudo snap install canonical-livepatch
sudo /snap/bin/canonical-livepatch enable kamikatkamikat
