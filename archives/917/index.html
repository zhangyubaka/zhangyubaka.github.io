为 Windows 交叉编译 Aria2-fast

Apr 28, 2017

AUR 上有一个 patch 去除了 Aria2 的最大线程限制.

https://aur.archlinux.org/packages/aria2-fast/

所以 Arch Linux 用户就可以直接 makepkg -si 了, Windows 怎么办？

自己交叉编译呗…

直接拿官方的 Dockerfile 改了下, 直接用就好了.

编译完之后就在 /aria2/src/aria2c.exe 中.
