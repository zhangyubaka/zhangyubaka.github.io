重装 CLT

Oct 1, 2016

更新了 Xcode 8.0 之后不知道为什么 App Store 同时推送了两个 Command Line Tools,一个是 Xcode 8.0 的,还有个 Xcode 7.3 的.

我:???

那么就重装吧.

sudo rm -rf /Library/Developer/CommandLineTools/
xcode-select --install

装完之后重开 App Store,应该就正常啦!
