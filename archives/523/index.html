去掉 Stargazer 代码框的背景色

May 8, 2016

如果你以前看过本博客,可以发现曾经的红白配色代码框没了,取而代之的是 highlightjs 的代码高亮.

什么?你没看到?该挂代理了朋友!

感谢 NAT 和 JackWorks 的帮助和提示,以及 grep 和 grep 的帮助,我总算把这东西砍了.

具体位置便是在 Stargazer 的 inc/custom-colors.php 里面.

$style .= "legend, mark, pre, .form-allowed-tags code { background-color: rgba( {$rgb}, 0.1 ); } ";

把这一行砍了就行.
