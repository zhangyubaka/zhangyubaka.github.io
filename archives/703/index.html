解决 Eix 没有数据的情况

Aug 12, 2016

在切换到了 Git 同步之后,eix-update 就抓不到数据了.于是对比了一下 rsync 和 git 同步的区别,发现 git 同步的缺少了 metadata,也就是 /usr/portage/metadata/md5-cache,原因很简单…因为开发者不需要这东西.

这种情况呢,我们换一个源,或者自己生成就好了.

换源:

sync-uri = https://github.com/gentoo-mirror/gentoo.git

自己生成:

egencache --jobs=9 --update --repo gentoo

PS:自己生成需要的时间很久…非常久.
